# GoLang Tutorial

A collection of small Go programs for learning core concepts like variables, arrays, slices, maps, functions, methods, interfaces, pointers, recursion, and more.

## Prerequisites

- Go installed on your system
- Check installation:

```bash
go version
```

## Run a Go file

Run any example directly:

```bash
go run helloworld.go
```

You can replace `helloworld.go` with any file from this project, for example:

```bash
go run enums.go
go run maps.go
go run interface.go
```

## Build and run executable

Build a binary:

```bash
go build -o app helloworld.go
```

Run the executable:

```bash
./app
```
